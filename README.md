#Install

`$ virtualenv env`

`$ source env/bin/activate`

`$ pip install -r requirements.txt `

`$ cp pro/config.cfg.sample pro/config.cfg `

`$ npm install gulp -g`

`$ npm install`

`$ gulp`


#Configs

* pro/config.cfg

* gulpfile.js

#Debug

`$ source env/bin/activate`

`$ python debug.py`


#Contribution

Check for a PEP-0008 compliance:

`$ pep8 --ignore=E501,E128 pro`
